# Medication Tracker

A web application that helps users manage and track their medications efficiently. Users can add, update, and monitor medication schedules through a simple and user-friendly interface.

## Features

- Add and manage medications
- Track medication schedules
- User-friendly dashboard
- Responsive design

## Tech Stack

- React.js
- Spring Boot
- MySQL
- CSS3
